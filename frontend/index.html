// how_signing_works
1. sign bytes = proto.Marshal(Transaction{...fields, signature:nil})
2. time field = Date.now() × 1000 (microseconds) — computed ONCE
3. same txTime value used in sign bytes AND tx body JSON
4. BLS12-381 G2 long signature (96 bytes) over sign bytes
5. address = SHA-256(pubKey).slice(0,20) — first 20 bytes
6. POST to /v1/tx (singular) with Content-Type: application/json
